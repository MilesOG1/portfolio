Space Evader (SIT151 Game Fundamentals)

A 2D arcade-style space shooter developed in Unity as part of my Game Fundamentals unit. Features include custom player movement, wave-based enemy spawning, and enhanced visual and audio feedback systems.

  • Mouse-based rotation and physiscs-based thrust control
  • Wave spawning and progressive difficulty system
  • Camera shake, explosions, and particle effects
  • Screen boundary clamping for polished player control

Virtual Maker Coop (SIT283 VR Development Challenge)

A collaborative VR maker environment developed using Unity’s XR Interaction Toolkit and Photon Fusion. The experience allows users to interact with tools, assemble virtual components, and configure system parameters in an immersive setting.

  • World-space UI start screen with controller interaction
  • Teleportation locomotion and object manipulation
  • Dynamic tool spawning and multi-step assembly sequence
  • Configurable workbench with speed and highlight controls
  • Autonomous conveyor simulation with sensory feedback
